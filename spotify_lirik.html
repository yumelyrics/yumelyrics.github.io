Spotify → Lirik

Paste link lagu Spotify atau ketik Artis - Judul. Lirik muncul kalau tersedia di database publik (berkorelasi dengan lagu yang ada liriknya di Spotify).

Masukkan Client ID dan Client Secret dari Spotify untuk mendapatkan metadata lagu yang lebih akurat (judul, artis, album art).

⚠️ Catatan penting: Spotify API resmi tidak menyediakan endpoint lirik. Lirik tetap diambil dari LRCLIB — database publik yang korelasinya baik dengan katalog Spotify.

Cara dapet credentials: 1. Buka developer.spotify.com/dashboard
2. Login → Create App
3. Isi nama app (bebas) → centang Web API → Create
4. Masuk ke Settings → copy Client ID dan Client Secret
5. Paste di bawah, klik Simpan.
