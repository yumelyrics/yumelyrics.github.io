Tentang Website Ini

13 Mei 2026 1 menit baca

Haiii

aku buat website ini buat alternatif ku terjemahin lagu, biasanya kan aku terjemahin lagu terus ku bikin video di youtube, tapi itu makan banyak waktu ku hanya untuk 1 video, kadang bisa 2 jam hanya untuk edit videonya aja. website ini aku coding sendiri dan dibantu oleh AI untuk error code nya. dan ini masih menggunakan Github untuk menjalankan websitenya, karena aku masih penerjemah lagu yang kecil.

terjemahanku ini tidak seperti terjemahan pada umumnya, aku mengubah arti dari kosakata asli agar sesuai dengan lirik lagunya. walaupun terlihat aneh untuk orang yang baru belajar bahasa jepang saat membaca terjemahannya, tapi memang begini dunia translator. terjemahan yang aku ubah, makna nya masih sama, cuman beda kosakata.

kalian bisa Request Lagu yang kalian suka untuk aku terjemahin, jika di internet tidak ada yg terjemahin lagunya. Free Ya.(soalnya aku pengen jadi translator, jadi aku pengen coba terjemahin lagu sebanyak mungkin.
