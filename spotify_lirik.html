Spotify → Lirik

Paste link lagu Spotify atau ketik Artis - Judul. Lirik muncul kalau tersedia di database publik (berkorelasi dengan lagu yang ada liriknya di Spotify).

Contoh:

—


    

    

      
    

  


  
  

    Cara pakai
    
    
      
  • Buka lagu di Spotify → Share → Copy link → paste di atas.
    • 
      
  • Hanya link track (bukan playlist/album/episode).
    • 
      
  • Kalau di Spotify lagu itu tidak ada liriknya, kemungkinan besar di sini juga tidak ketemu.
    • 
      
  • Lirik diambil dari LRCLIB — database lirik publik.
    • 
      
  • Spotify API credentials opsional — kalau tidak diisi, metadata diambil dari oEmbed (kurang akurat).
    •
