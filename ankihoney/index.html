2 Pilih Note Type & Field

✨ Background Kartu Opsional

Ganti background tampilan kartu di Anki pakai foto sendiri. Berlaku buat note type - yang lagi dipilih di atas.

⚠️ Penting kalau note type ini udah dipakai di deck lain yang udah ada duluan di Anki kamu (misal note type yang sama dipakai bareng-bareng di banyak deck): Anki cuma nge-update tampilan note type pas import kalau ada perubahan baru yang kedeteksi. Tool ini udah otomatis nanganin itu, tapi kalau setelah import background tetep gak muncul, biasanya karena Anki nyimpen versi lama note type-nya. Coba hapus dulu deck versi lama dari Anki (atau note type-nya lewat Manage Note Types) sebelum import ulang file yang baru, biar gak bentrok sama versi lama.
🖼️ Pilih atau seret foto ke sini JPG/PNG, otomatis dikompres biar file .apkg tetap ringan
✅ Foto ikut kesimpen di dalam .apkg, jadi tetep muncul walau offline. Kalau note type ini udah kepake di deck lain yang lebih lama di Anki kamu, baca peringatan kuning di atas dulu ya.
⚠️ Foto harus tetap online selamanya biar background gak ilang pas dipelajari. Gambar juga gak ikut kesimpen ke dalam .apkg (jadi butuh internet nyala pas belajar di Anki), tapi cara ini paling anti-gagal karena gak nyentuh media/note type sama sekali.
